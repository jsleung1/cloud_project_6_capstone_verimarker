Overview: VeriMarker

This is a demo revamp project for VeriGuide, a plagiarism_detection system used to check similarities on documents from different sources to uphold academic honesty. Using this site, students can upload files to their course assignments, and instructors can create courses and assignments, as well as manage students' submissions.

This site currently supports Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge

As part of the process to educate students about academic honesty and proper referencing, please visit The Chinese University of Hong Kong: Honesty in Academic Work for more information.
